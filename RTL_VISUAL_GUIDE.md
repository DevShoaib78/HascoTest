## Key RTL Differences

### Text Direction
- **English**: Left → Right
- **Arabic**: Right ← Left

### Element Positioning
| Element | English | Arabic |
|---------|---------|--------|
| Logo | Left | Right |
| Navigation | Center-Right | Center-Left |
| Language Button | Far Right | Far Left |
| Form Labels | Above/Left | Above/Right |
| Submit Button | Left | Right |
| Icons | Before text | After text |
| Arrows | → | ← |

### Flex Direction
```css
/* English */
flex-direction: row;

/* Arabic */
flex-direction: row-reverse;
```

### Text Alignment
```css
/* English */
text-align: left;

/* Arabic */
text-align: right;
```

### Padding/Margin
```css
/* English */
padding-left: 1rem;

/* Arabic */
padding-right: 1rem;
```

---

## Animation Differences

### Slide In
**English**: Slides from left → right  
**Arabic**: Slides from right ← left

### Underline Growth
**English**: Grows from left → right  
**Arabic**: Grows from right ← left

### Carousel Movement
**English**: Next = →, Previous = ←  
**Arabic**: Next = ←, Previous = →

---

## Typography Differences

### Font Stack
**English**: Montserrat, Helvetica, Arial  
**Arabic**: Segoe UI, Tahoma, Arial (optimized for Arabic)

### Letter Spacing
**English**: Normal to wide tracking  
**Arabic**: Normal (no extra tracking)

### Font Weight
**English**: Light (300) to Bold (700)  
**Arabic**: Medium (500) to Bold (700) for better readability

## Testing Visual Checklist

### English (LTR)
- [ ] Logo on left
- [ ] Menu items flow left to right
- [ ] Language button on right shows "عربي"
- [ ] Text aligns left
- [ ] Arrows point right (→)
- [ ] Forms align left

### Arabic (RTL)
- [ ] Logo on right
- [ ] Menu items flow right to left
- [ ] Language button on left shows "English"
- [ ] Text aligns right
- [ ] Arrows point left (←)
- [ ] Forms align right

---

**Note**: All visual differences are automatically handled by the `dir="rtl"` attribute and CSS adjustments. No manual intervention needed for most elements.
